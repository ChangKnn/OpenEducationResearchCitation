# 《开放教育研究》引用格式

通过定义Locale标签实现对中英文双语文献引用样式的定义。

期刊要求引用格式如下：

中文引用：

```text
胡丹妮，章梦瑶，郑勤华（2019）. 基于滞后序列分析法的在线学习者活动路径可视化分析[J]. 电化教育研究， （5）：55-63.
```

英文引用：

```text
Papanikolaou，K.，Makri，K.，& Roussos，P.（2017）. Learning design as a vehicle for developing TPACK in blended teacher training on technology enhanced learning[J]. International Journal of Educational Technology in Higher Education，14（1）：34-48.
```

## 注意

由于使用Locale标签，因此对于英文文献需要在Zotero中设置语言为 `en-US en-us en english English`其中之一，进而实现双语功能。
